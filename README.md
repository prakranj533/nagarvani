# NagarVaani

A citizen feedback platform for India that connects citizens with their government representatives (MLA, MP, Local Authorities) and aggregates local news.

## Features

- **Submit Feedback**: Multi-step form with representative lookup by pincode
- **Track Status**: Track complaint status with timeline
- **Community Issues**: View and upvote local issues
- **Local News**: Hyperlocal news sharing platform
- **Aggregated News**: Real-time news from multiple sources across India
- **File Attachments**: Upload photos, videos, documents with feedback
- **Location-based**: Personalized content based on your pincode

## Tech Stack

- Next.js 16 with TypeScript
- Tailwind CSS
- Lucide React Icons
- In-memory data store (demo mode)

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)
